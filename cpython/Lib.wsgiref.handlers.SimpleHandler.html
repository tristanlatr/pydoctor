class Lib.wsgiref.handlers.SimpleHandler(BaseHandler):

Known subclasses: Lib.wsgiref.handlers.BaseCGIHandler

View In Hierarchy

Handler that's just initialized with streams, environment, etc. This handler subclass is intended for synchronous HTTP/1.0 origin servers, and handles sending the entire response output, given the correct inputs. Usage:: handler = SimpleHandler( inp,out,err,env, multithread=False, multiprocess=True ) handler.run(app)
