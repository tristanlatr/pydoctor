Lib.logging.handlers.DatagramHandler

class Lib.logging.handlers.DatagramHandler(SocketHandler):

View In Hierarchy

A handler class which writes logging records, in pickle format, to a datagram socket. The pickle which is sent is that of the LogRecord's attribute dictionary (__dict__), so that the receiver does not need to have the logging module installed in order to process the logging event. To unpickle the record at the receiving end into a LogRecord, use the makeLogRecord function.
