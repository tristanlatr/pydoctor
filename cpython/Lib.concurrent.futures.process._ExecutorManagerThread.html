class Lib.concurrent.futures.process._ExecutorManagerThread(threading.Thread):

View In Hierarchy

Manages the communication between this process and the worker processes. The manager is run in a local thread. Args: executor: A reference to the ProcessPoolExecutor that owns this thread. A weakref will be own by the manager as well as references to internal objects used to introspect the state of the executor.
