class Lib.formatter.NullWriter:

Known subclasses: Lib.formatter.AbstractWriter, Lib.formatter.DumbWriter

View In Hierarchy

Minimal writer interface to use in testing & inheritance. A writer which only provides the interface definition; no actions are taken on any methods. This should be the base class for all writers which do not need to inherit any implementation methods.
