class Lib.http.cookiejar.CookiePolicy:

Known subclasses: Lib.http.cookiejar.DefaultCookiePolicy

View In Hierarchy

Defines which cookies get accepted from and returned to server. May also modify cookies, though this is probably a bad idea. The subclass DefaultCookiePolicy defines the standard rules for Netscape and RFC 2965 cookies -- override that if you want a customized policy.

Method set_ok Return true if (and only if) cookie should be accepted from server.
Method return_ok Return true if (and only if) cookie should be returned to server.
Method domain_return_ok Return false if cookies should not be returned, given cookie domain.
