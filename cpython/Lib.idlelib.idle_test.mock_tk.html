Lib.idlelib.idle_test.mock_tk

module documentation

Classes that replace tkinter gui objects used by an object being tested. A gui object is anything with a master or parent parameter, which is typically required in spite of what the doc strings say.

Class Event Minimal mock with attributes for testing event handlers.
Class Var Use for String/Int/BooleanVar: incomplete
Class Mbox_func Generic mock for messagebox functions, which all have the same signature.
Class Mbox Mock for tkinter.messagebox with an Mbox_func for each function.
Class Text A semi-functional non-gui replacement for tkinter.Text text editors.
Class Entry Mock for tkinter.Entry.
