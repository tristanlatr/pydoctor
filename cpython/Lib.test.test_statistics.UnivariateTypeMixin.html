Mixin class for type-conserving functions. This mixin class holds test(s) for functions which conserve the type of individual data points. E.g. the mean of a list of Fractions should itself be a Fraction. Not all tests to do with types need go in this class. Only those that rely on the function returning the same type as its input data.
